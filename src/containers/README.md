# React-Native Redux TODO App

## Features

1. Add Task
2. Swipe left to delete
3. Mark Completed
4. Pending & Completed Task Views

### Steps to use

```bash
git clone https://github.com:Piyushhbhutoria/reduxTodoApp.git
cd reduxTodoApp
yarn install
yarn start
```
